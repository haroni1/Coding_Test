## 💡 문제 이해

> 문제를 내가 이해한 방식으로 한 줄 요약

- **문제:**  
    두 개의 정수 `A`와 `B`가 입력으로 주어질 때, **A - B 의 결과를 출력하는 문제**
    

---

## 🚀 접근 방식 (Logic)

1. **입력 받기**
    
    - 한 줄에 `A B` 형태로 들어오기 때문에 `Split()` 필요.
        
2. **자료형 변환**
    
    - `Console.ReadLine()` → `string`
        
    - `int.Parse()` 로 정수 변환
        
3. **계산**
    
    - 단순하게 `a - b`
        
4. **출력**
    
    - `Console.WriteLine()` 사용
        
5. **시간 복잡도**
    
    - 연산 1번 → `O(1)` (가장 단순한 수준)
        

---

## 💻 정답 코드 (C#)

``` C#
using System;

class Program
{
    static void Main()
    {
        string[] input = Console.ReadLine().Split();
        int a = int.Parse(input[0]);
        int b = int.Parse(input[1]);

        Console.WriteLine(a - b);
    }
}
```

## 📝 오답 노트 & 회고

- **틀린 이유(초반 생각):**
    
    - 입력이 줄바꿈으로 들어온다고 착각해서 `Console.ReadLine()`을 두 번 사용했음.
        
- **배운 점:**
    
    - 백준에서는 **입력 형식이 중요**하며,  
        `A B` 처럼 **한 줄에 공백으로 주어지는 경우가 많다.**
        
    - 입력 예시를 무조건 정확히 보고 처리 방식 고려해야 한다.
        
- **다른 방식 참고 포인트:**
    
    - `Convert.ToInt32()` 사용해도 가능하지만  
        `int.Parse()`가 더 일반적.
        
    - 테스트가 매우 단순하므로 무조건 직관적으로 처리하면 됨.
