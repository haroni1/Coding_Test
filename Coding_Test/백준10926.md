## 💡 문제 이해

> 문제를 내가 이해한 방식으로 한 줄 요약

- **문제:**  
    입력된 아이디가 이미 존재하므로,  
    그 아이디 뒤에 `"??!"` 를 붙여서 출력하는 문제.
    

---

## 🚀 접근 방식 (Logic)

1. **입력 받기**
    
    - 아이디는 문자열 형태로 단일 입력된다.
        
    - 공백이 없고, 길이는 최대 50자.
        
2. **가공**
    
    - 입력된 문자열 뒤에 `"??!"`를 그대로 이어 붙인다.
        
3. **출력**
    
    - 가공된 문자열을 출력한다.
        
4. **시간 복잡도**
    
    - 문자열 처리 한 번 → `O(n)` (여기서 n은 문자열 길이)
        

---

## 💻 정답 코드 (C#)

```C#
using System;

class Program
{
    static void Main()
    {
        string input = Console.ReadLine();
        Console.WriteLine(input + "??!");
    }
}
```
---

## 📝 오답 노트 & 회고

- **틀린 이유(초반 생각):**
    
    - `Console.ReadLine()`이 문자열을 반환하는데,  
        배열로 받으려다 `string[]` 타입 오류가 발생함.
        
    - `Split()`이 필요하다고 생각했지만,  
        입력이 단일 문자열이므로 필요하지 않았다.
        
- **배운 점:**
    
    - 입력 형식을 먼저 정확히 이해하고,  
        불필요한 자료 구조를 만들지 않는 것이 중요하다.
        
    - `string`과 `string[]`의 차이를 다시 정리할 수 있었다.
        
- **다른 방식 참고 포인트:**
    
    - `string.Concat(input, "??!")` 또는 `$"{input}??!"` 등의 방법도 가능하지만,  
        `+` 연산이 가장 직관적이고 문제 조건에 충분히 적합하다.
