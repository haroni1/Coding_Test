## 💡 문제 이해

> 문제를 내가 이해한 방식으로 한 줄 요약

- **문제:**  
    입력된 **불기(BE) 연도**를 **서기(AD)** 연도로 변환하여 출력하는 문제.  
    (공식: `서기 = 불기 - 543`)
    

---

## 🚀 접근 방식 (Logic)

1. **입력 받기**
    
    - `Console.ReadLine()`을 사용하여 문자열로 입력된다.
        
    - 불기 연도는 정수이므로 `int`로 변환이 필요하다.
        
2. **가공**
    
    - 정수 변환 후 `543`을 빼준다.
        
3. **출력**
    
    - 계산된 결과를 출력한다.
        
4. **시간 복잡도**
    
    - 단순 산술 연산 1회 → `O(1)`
        

---

## 💻 정답 코드 (C#)
```C#
using System;

class Program
{
    static void Main()
    {
        int y = int.Parse(Console.ReadLine());
        Console.WriteLine(y - 543);
    }
}
```

---

## 📝 오답 노트 & 회고

- **처음 실수한 부분**
    
    - `Console.ReadLine()`은 **string**을 반환하는데 바로 산술 연산을 시도함.
        
    - `"2541" - 543` 같은 형태는 컴파일 오류 발생.
        
- **배운 점**
    
    - **자료형 변환(int.Parse)**이 필요한지 판단하는 습관이 중요함.
        
    - 입력 형식과 요구 결과를 먼저 이해하면 코드가 더 깔끔해짐.
        
- **개선 가능 포인트**
    
    - 다음부터는 `int.TryParse()` 같은 안정적인 입력 처리도 고려 가능.
        
    - 한 줄짜리 코드로도 표현 가능:
        

`Console.WriteLine(int.Parse(Console.ReadLine()) - 543);`
