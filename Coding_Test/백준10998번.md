## 💡 문제 이해

> 문제를 내가 이해한 방식으로 한 줄 요약

- **문제:**  
    두 개의 정수 `A`와 `B`가 입력으로 주어질 때, **A x B 의 결과를 출력하는 문제**
    

---

## 🚀 접근 방식 (Logic)

1. **입력 받기**
    
    - 한 줄에 `A B` 형태로 들어오기 때문에 `Split()` 필요.
        
2. **자료형 변환**
    
    - `Console.ReadLine()` → `string`
        
    - `int.Parse()` 로 정수 변환
        
3. **계산**
    
    - 단순하게 `a x b`
        
4. **출력**
    
    - `Console.WriteLine()` 사용
        
5. **시간 복잡도**
    
    - 연산 1번 → `O(1)` (가장 단순한 수준)
        

---

## 💻 정답 코드 (C#)

``` C#
using System;

class Program
{
    static void Main()
    {
        string[] input = Console.ReadLine().Split();
        int a = int.Parse(input[0]);
        int b = int.Parse(input[1]);

        Console.WriteLine(a * b);
    }
}
```
