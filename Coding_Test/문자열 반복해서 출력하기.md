## 💡 문제 이해

> 문제를 나만의 언어로 요약해서 적어보세요.

- **문제**: 문자열 1개와 정수 1개를 입력받아서, **문자열을 정수만큼 이어 붙여서 한 줄로 출력하는 문제**
    
    - 예) 입력: `for 3` → 출력: `forforfor`
        
- **핵심 조건:**
    
    - 입력은 **한 줄에 문자열과 숫자가 공백으로 주어진다**
        
        - 예: `"for 3"`
            
    - 출력은 **줄바꿈 없이** 문자열이 **연속해서** 나와야 한다
        
        - `forforfor` (줄바꿈 X, 공백 X)
            
    - 정수는 반복 횟수이므로 음수는 나오지 않는다 (문제 기본 조건)
        

---

## 🚀 접근 방식 (Logic)

> 코드를 짜기 전, 어떤 알고리즘과 자료구조를 쓸지 설계하는 단계입니다.

1. **입력받기**
    
    - `Console.ReadLine()`으로 한 줄 전체를 입력받는다.  
        예: `"for 3"`
        
2. **입력 파싱**
    
    - `.Split()`으로 공백 기준 분리  
        → `input[0] = "for"`, `input[1] = "3"`
        
    - `input[0]`은 그대로 **문자열(string)**
        
    - `input[1]`은 `int.Parse()`로 **정수로 변환**해서 반복 횟수로 사용  
        → `int n = int.Parse(input[1]);`
        
3. **문자열 반복 출력**
    
    - `for`문으로 `0 ~ n-1`까지 반복하면서 `Console.Write()`를 사용해 문자열을 이어 붙여서 출력
        
    - `Console.WriteLine()`을 쓰면 매번 줄바꿈이 들어가서  
        `for`  
        `for`  
        `for`  
        이렇게 나오므로 **반드시 `Write` 사용**
        
4. **시간 복잡도**
    
    - 문자열 길이를 `L`, 반복 횟수를 `n`이라고 하면  
        → 문자열 출력 연산: `O(n * L)`
        
    - n과 L이 매우 작으므로 사실상 **O(1) 취급 가능**

## 💻 정답 코드 (C#)

``` c#
using System;

class Program
{
    static void Main()
    {
        string[] input = Console.ReadLine().Split();

        string s = input[0];           // 반복할 문자열
        int n = int.Parse(input[1]);   // 반복 횟수

        for (int i = 0; i < n; i++)
        {
            Console.Write(s);          // 줄바꿈 없이 이어서 출력
        }
    }
}
```

## 📝 오답 노트 & 회고

> 틀렸다면 왜 틀렸는지, 맞았다면 더 좋은 방법은 없는지 기록하세요.

- **틀린 이유 / 헷갈렸던 부분**
    
    - 처음에는 `for (a>0; a--)` 같이 `for`문 문법을 잘못 썼다
        
        - C#에서 `for`는 항상 `for (초기식; 조건식; 증감식)` 형태여야 한다
            
    - `Console.WriteLine()`을 써서
        
        - `for`  
            `for`  
            `for`  
            이렇게 줄바꿈된 출력이 나왔고,  
            문제에서 원하는 `forforfor` 형식과 맞지 않았다
            
- **배운 점**
    
    - 콘솔 출력에서 **줄바꿈 여부**에 따라 `Write` / `WriteLine`을 구분해서 써야 한다
        
        - `WriteLine()` → 출력 후 자동 줄바꿈
            
        - `Write()` → 줄바꿈 없이 이어붙이기
            
    - 입력은 항상 **문자열로 들어오고**, 숫자가 필요하면 꼭 `int.Parse()` 같은 변환을 거쳐야 한다
        
    - 반복문 문법을 정확히 기억할 것:
        
        `for (초기식; 조건식; 증감식) { ... }`
        
        잘못된 예: `for (a>0; a--)` (초기식/조건식/증감식 구분이 없음)
        
- **다른 방식 참고**
    
    - C#에서는 굳이 반복문을 쓰지 않고, `new string()`이나 `String.Concat(Enumerable.Repeat(...))` 같은 방법도 있지만
        
        - 프로그래머스 LV0 난이도에서는 **기본 문법 연습용이라 `for`문으로 푸는 게 더 교육적**임.
