## 💡 문제 이해

> 문제를 내가 이해한 방식으로 한 줄 요약

- **문제:**  
    두 자연수 `A`와 `B`가 주어질 때,  
    **A+B, A-B, A×B, A÷B(몫), A%B(나머지)** 를  
    각각 줄바꿈하여 출력하는 문제.
    

---

## 🚀 접근 방식 (Logic)

1. **입력 받기**
    
    - `"A B"` 형태로 입력되므로 `Split()`을 사용해 분리한다.
        
2. **자료형 변환**
    
    - 정수 계산이므로 `int.Parse()`로 정수형으로 변환한다.
        
3. **계산**
    
    - 다섯 가지 연산을 수행한다:  
        `+`, `-`, `*`, `/`(정수 나눗셈), `%`
        
4. **출력**
    
    - 계산된 결과를 각 줄에 하나씩 순서대로 출력한다.
        
5. **시간 복잡도**
    
    - 단순 연산 5번 → `O(1)`
        

---

## 💻 정답 코드 (C#)

```C#
using System;

class Program
{
    static void Main()
    {
        string[] input = Console.ReadLine().Split();
        int a = int.Parse(input[0]);
        int b = int.Parse(input[1]);

        Console.WriteLine(a + b);
        Console.WriteLine(a - b);
        Console.WriteLine(a * b);
        Console.WriteLine(a / b);
        Console.WriteLine(a % b);
    }
}

```

## 📝 오답 노트 & 회고

- **틀린 이유(초반 생각):**
    
    - 처음에는 곱셈 연산만 수행하는 문제라고 생각했지만,  
        문제를 다시 읽어보니 다섯 개의 연산을 모두 해야 한다는 조건이 있었다.
        
- **배운 점:**
    
    - 연산 문제가 반복될수록 단순히 연산식만 보는 것이 아니라  
        **출력 형식과 요구 연산 개수를 꼼꼼하게 확인해야 한다는 점**을 깨달았다.
        
- **다른 방식 참고 포인트:**
    
    - `Convert.ToInt32()`를 사용해도 되지만  
        `int.Parse()`가 더 명확하고 일반적으로 사용된다.
        
    - 출력 형식이 줄바꿈 조건일 경우,  
        `Console.WriteLine()`을 활용하면 자동 처리되어 편리하다.
